# 坦克大战 🎮

> **本仓库代码由 AI（Claude Code）生成**

基于 Python + Pygame 的经典 Battle City 风格坦克对战游戏。

## 快速开始

```bash
pip install pygame
python main.py
```

## 操作

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动坦克 |
| J / 空格 | 射击 |
| Enter | 开始游戏 |
| R | 重新开始 |
| Q  | 返回标题 |

## 游戏规则

- **B 方（玩家/绿色）**：1 辆坦克，3 条命，守卫己方主将
- **A 方（AI/红色）**：每关 20 辆坦克，每次最多 4 辆同时活跃
- 摧毁敌方全部坦克或敌方主将即可获胜
- 注意保护己方主将，被摧毁则游戏结束

## 项目结构

```
tank_battle/
├── main.py        # 入口 + 游戏状态机
├── const.py       # 常量定义
├── core/          # 状态机框架
├── entity/        # 游戏实体（坦克、子弹、主将、道具）
├── map/           # 地图系统 + 关卡数据
├── ui/            # HUD 和菜单画面
└── assets/        # 资源文件（关卡配置等）
```
